# 후기

- `react-hook-form`은 기본적으로 react의 `ref`를 이용한다.
- validation check 로직은 react-hook-form을 이용하고
- UI 단은 손쉽게 커스터마이징 해서 사용이 가능하다!
- next.js와 결합에 문제가 없는지 확인이 필요하다.
  - 이건 회사 프로젝트에서 제대로 돌아가는지 먼저 확인이 필요하다.
